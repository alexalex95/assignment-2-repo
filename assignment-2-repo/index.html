How to Apply CSS to HTML

You can include an external CSS file by using a <link> tag inside the <head> of your HTML file: 

<link rel="stylesheet" href="style.css">

How to Use JavaScript in a Webpage

To link an external JavaScript file, use the <script> tag at the end of your <body> tag: 

<script src="script.js"></script>
